# 👟 Nike — Адаптивная верстка и интерактивный фронтенд магазина кроссовок

🔗 [Посмотреть Live Demo](https://bibapirat6.github.io/Nike/)

## 🎯 О проекте и цель разработки

Этот проект — фронтенд-приложение многостраничного интернет-магазина кроссовок Nike, разработанное мной в рамках производственной практики в ИТ-компании «Арбис». Вместо использования готовых шаблонов или тяжелых UI-фреймворков, весь интерфейс и бизнес-логика клиентской части были написаны с нуля на чистом технологическом стеке.

Главная цель разработки — закрепить навыки коммерческой адаптивной верстки, интеграции кастомной типографики, а также реализации динамического пользовательского интерфейса (интерактивная корзина, управление состоянием товаров и симуляция оформления заказа) на чистом JavaScript без сторонних библиотек.

## 🛠️ Какие навыки я прокачал на этом проекте

* **DOM-манипуляции и State Management:** Реализовал полноценную логику корзины на Vanilla JS (добавление, динамическое удаление, пересчет итоговой стоимости) с обновлением состояния интерфейса в реальном времени.
* **Коммерческий UI/UX и Работа с типографикой:** Интегрировал кастомные шрифты (семейство Helvetica Neue Cyr), обеспечив их правильное сглаживание, иерархию и корректное отображение на любых устройствах.
* **Оптимизация и модульность:** Организовал понятную структуру проекта с разделением логики (основной интерфейс и выделенная страница `cart.html`), а также подготовил минифицированные стили `style.min.css` для ускорения загрузки страниц.
* **Работа в условиях реальной практики:** Прошел полный цикл от разбора требований до готовой фронтенд-сборки под кураторством наставника из веб-студии.

## 💻 Технологический стек

* **HTML5:** Семантическая разметка многостраничного сайта, правильная иерархия тегов, независимая структура страниц `index.html` и `cart.html`.
* **CSS3:** Flexbox и Grid Layout для построения адаптивных сеток, кастомные анимации UI-элементов, оптимизированный и минифицированный файл стилей `style.min.css`.
* **Vanilla JavaScript:** Обработка событий, динамическое изменение DOM-дерева, управление состоянием корзины и симуляция процесса оплаты.
* **Шрифты:** Helvetica Neue Cyr.

## 🚀 Инструкция по локальному запуску

Для корректной работы интерактивных функций, путей к ассетам и переходов между страницами проект рекомендуется запускать через локальный веб-сервер.

### 1. Подготовка (Клонирование репозитория)

Откройте терминал (в Linux / macOS) или Git Bash (в Windows) и выполните команды:

```bash
# Клонируем проект
git clone [https://github.com/BibaPirat6/Nike.git](https://github.com/BibaPirat6/Nike.git)

# Переходим в папку проекта
cd Nike
```

## 2. Запуск через Visual Studio Code (Live Server)

Если вы используете VS Code:

1. Установите расширение **Live Server** (автор: Ritwick Dey) через вкладку Extensions (Ctrl+Shift+X).
2. Откройте папку проекта в редакторе.
3. Кликните правой кнопкой мыши по файлу `index.html` и выберите **"Open with Live Server"**.
4. Сайт автоматически откроется в браузере по адресу `http://127.0.0.1:5500`.
